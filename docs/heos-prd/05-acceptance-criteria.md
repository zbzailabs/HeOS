# HeOS 一期验收标准

版本日期：2026-06-08

## 1. 验收范围

一期验收覆盖完整平台基础能力和腾龙小学智慧农场首个真实设备接入场景。验收以真实租户、真实项目、真实设备接口、真实页面流程和可复现命令为准。

## 2. 业务闭环

|验收项|通过标准|
|---|---|
|统一工作台|用户登录后进入工作台，查看项目、设备、告警、任务和近期同步状态。|
|项目管理|创建或导入项目、站点、地块、大棚和设备台账。|
|作物模型|创建作物模型，配置阶段、指标、风险和农事模板。|
|农事任务|按作物阶段生成任务，记录执行、照片、定位和验收结果。|
|告警中心|阈值、离线和供应商告警进入列表，支持确认、处理和关闭。|
|追溯展示|作物周期关联地块、农事、投入品、检测、采收和公开展示资料。|
|AI 辅助|AI 回答只引用授权数据，展示来源并写入审计记录。|

## 3. 权限隔离

|验收项|通过标准|
|---|---|
|租户隔离|A 租户无法读取、搜索、导出或通过 AI 查询 B 租户数据。|
|项目范围|项目负责人只能访问授权项目。|
|只读角色|监管查看用户、访客和教学查看用户无法修改业务数据。|
|控制权限|无控制权限用户无法创建 `ControlCommand`。|
|审计记录|登录、导出、AI 调用、设备控制、策略变更写入 `AuditLog`。|

## 4. 仁科设备接入

|验收项|通过标准|
|---|---|
|登录|使用 `http://api.farm.0531yun.cn` 登录接口获取 token。|
|设备列表|读取设备列表并找到设备 `40406816`。|
|设备类型|以接口返回的 `deviceType` 决定历史数据接口。|
|实时数据|展示设备最新指标、采样时间和供应商状态。|
|在线判断|最后一次数据时间超过 5 分钟判定离线。|
|历史曲线|按单个指标查看历史曲线。|
|基础告警|阈值超限、离线、供应商告警进入 `Alert`。|
|同步异常|鉴权失败、参数错误、接口异常写入同步异常记录。|

## 5. 性能

|验收项|通过标准|
|---|---|
|工作台首屏|核心内容 3 秒内完成加载。|
|项目详情|项目概览、设备状态、告警摘要 2 秒内返回。|
|列表查询|设备、告警、任务列表支持分页和服务端筛选。|
|历史曲线|单设备单指标 7 天历史数据查询 3 秒内返回。|
|构建|`pnpm run build` 成功完成。|

## 6. 安全

|验收项|通过标准|
|---|---|
|密钥管理|数据库连接、供应商密码、token、AI Key 不进入前端代码。|
|服务端访问|供应商接口只由服务端调用。|
|控制命令|控制功能默认关闭；打开前具备权限、确认、阈值、审批、回执和审计链路。|
|AI 边界|AI 不直接触发设备控制。|
|公开页面|追溯公开页只展示允许公开的数据。|

## 7. 观测与运维

|验收项|通过标准|
|---|---|
|请求日志|关键接口记录 trace id、用户、租户、耗时和结果。|
|同步日志|每次仁科同步记录开始时间、结束时间、设备数、成功数、失败数。|
|告警日志|告警产生、确认、处理和关闭记录完整。|
|AI 日志|AI 调用记录模型、输入摘要、来源、输出摘要、成本和调用人。|
|部署记录|Cloudflare Workers 部署保留版本、时间和构建结果。|

## 8. TanStack 与 Cloudflare 优先级

|验收项|通过标准|
|---|---|
|应用运行|Web 管理端运行在 TanStack Start + Cloudflare Workers。|
|业务数据|新增一期业务表优先使用 Cloudflare D1；外部 PostgreSQL/Neon 使用前有技术评审记录。|
|对象资料|图片、报告、追溯素材和导出文件进入 Cloudflare R2。|
|配置缓存|租户配置、字典、功能开关和低频缓存进入 Cloudflare KV。|
|异步任务|仁科同步、告警派发、报告生成和失败重试进入 Cloudflare Queues。|
|定时任务|定时拉取和补偿同步使用 Cloudflare Cron Triggers。|
|密钥管理|供应商凭据、AI Key 和生产密钥使用 Workers Secrets 或部署平台密钥。|

## 9. 开发验证

每次交付运行：

```bash
pnpm test
pnpm run build
```

文档变更运行：

```bash
rg -n "^##\\s*$|^#\\s*$" docs
rg -n "Next\\.js|NestJS|Supabase|Drizzle|TimescaleDB|Dokploy monorepo" docs
```

第一条命令无结果，表示没有空标题。第二条命令只允许在“后续评估项”、历史说明或例外方案中出现相关技术，不得把这些技术写成当前一期默认主栈。
