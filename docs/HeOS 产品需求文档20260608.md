# HeOS 产品需求文档与开发指南

版本日期：2026-06-08

本文档是 HeOS 一期产品与工程文档入口。原始混合草稿已经拆分为产品需求、技术架构、开发指南、仁科设备接入和验收标准五份分册。各分册共同作为 HeOS 一期研发、评审、验收和交付依据。

## 1. 产品定位

HeOS 是面向农业科技企业、合作社、园区、学校农场和政府协同场景的作物种植综合服务与智能化管理平台。平台围绕作物模型、多租户运营、设备接入、风险告警、农事管理、追溯展示和 AI 辅助服务建设统一工作台。

一期采用完整平台边界，覆盖统一工作台、租户权限、项目和基地资产、地块和大棚、设备台账、作物模型、农事任务、告警中心、追溯档案、AI 辅助、系统管理。腾龙小学智慧农场的仁科设备接入作为首个真实场景，验证真实设备数据、实时监测、历史曲线和基础告警链路。

## 2. 当前代码状态

当前仓库是 TanStack Start 初始化项目，不是完整业务平台。现有代码事实如下：

- Web 框架：TanStack Start、TanStack Router、React 19、Vite。
- 样式体系：Tailwind CSS 4、shadcn/ui 相关依赖、Lucide React。
- 数据库示例：Prisma 7、Neon serverless PostgreSQL；当前 schema 仅包含 `Todo` 示例模型。该部分是初始化示例，不作为一期业务默认数据底座。
- Cloudflare 运行底座：Cloudflare Workers / Wrangler，入口为 `@tanstack/react-start/server-entry`。
- 测试：Vitest，当前脚本使用 `--passWithNoTests`。
- 包管理：pnpm。

本文档中的业务模块、实体和接口属于一期目标设计。实现过程中必须先写规格、验收标准和边界，再编写代码。

## 4. 一期边界

一期交付完整平台基础能力，同时以首个真实设备场景验证技术链路。

### 4.1 平台能力

- 统一工作台：项目总览、角色入口、待办、告警、近期数据。
- 租户权限：租户、组织、用户、角色、数据范围、审计。
- 资产管理：项目、基地、站点、地块、大棚、设备、点位模板。
- 作物模型：作物品类、品种、阶段、指标、风险、农事模板。
- 农事管理：任务、执行记录、投入品、照片、定位、验收。
- 告警中心：阈值、离线、设备异常、控制失败、处理闭环。
- 追溯展示：作物周期、农事记录、投入品、检测、采收和公开展示。
- AI 辅助：问答、告警解释、农事建议、报告摘要、来源引用、审计记录。
- 系统管理：字典、配置、供应商账号、密钥、日志、版本记录。

### 4.2 首个真实场景

腾龙小学智慧农场作为一期首个真实接入场景，完成以下链路：

1. 使用仁科接口登录并获取 token。
2. 读取用户设备列表，确认设备 `40406816` 的真实类型。
3. 拉取实时数据并写入最新值。
4. 按设备类型读取历史数据。
5. 计算在线状态，超过 5 分钟未更新判定为离线。
6. 展示项目页、设备状态、实时数据、历史曲线和基础告警。
7. 控制功能默认关闭，只保留权限、审计、审批和安全阈值设计。

## 5. 关键决策

- 一期新功能优先采用 TanStack + Cloudflare 生态：TanStack Start、TanStack Router、TanStack Query、TanStack Table、TanStack Form、React 19、Vite、Tailwind CSS 4、Cloudflare Workers、D1、R2、KV、Queues、Cron Triggers、Workers Bindings、Wrangler、pnpm。
- 当前仓库中的 Prisma/Neon 只作为初始化示例和兼容参考。业务主数据优先进入 Cloudflare D1；对象资料进入 R2；配置、字典、功能开关和低频缓存进入 KV；仁科同步、告警派发和异步任务使用 Queues 与 Cron Triggers。
- 外部 PostgreSQL、Prisma、Neon 只作为例外方案：当 D1 无法满足容量、复杂查询、事务模型或迁移兼容性要求时，经技术评审后引入。
- Next.js、NestJS、Taro、Supabase、Drizzle、TimescaleDB、Dokploy monorepo 不写入当前一期主技术栈；这些内容只作为后续评估项保留在技术架构分册。
- AI 只作为辅助能力进入一期，不直接触发设备控制。
- 设备控制属于高风险能力，进入权限校验、二次确认、安全阈值、审批、回执和审计链路。
- 仁科接入分册保留账号、密码、设备编号和接口域名原文。该分册属于内部资料，不适合公开发布。
